Node Module

Node module is a block of code that we can import and use in our code.It is reusable and prevents writing long files.We can create our own module , it can also be installed using npm available third party libraries they are also available in node(pre installed).When we install some package module is installed in folder named node module/.

Module system

In the begining when node js was created there was no module system for developers.Module system means to split , reuse and import module.

Commonjs

Commonjs is a module system that was created to solve the problem of spliting sharing and reusing code.It used require to import module in code.It works in synchronous way.As in the begining there was no modules system so nodejs used commonjs as native module system.It does not work with browsers.

Ecmascript (ES6)

Later on Ecmascript was introduced as a new module system.It uses import/export to use module on server side.It works asynchronously.It also works with browser.It is supported with modern nodejs but commonjs also works with node because millions of packages are in commonjs.

    Types of exports in ES6
  • Named export

  • Default export
